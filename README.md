🎲 DiceMaster Pro — генератор случайных чисел (игральные кости)
Универсальный генератор игральных костей с поддержкой различных типов костей, весовых коэффициентов, истории бросков, статистики и красивой анимации в консоли.
Реализован на 7 языках программирования для демонстрации работы с генераторами случайных чисел и пользовательским интерфейсом.

🧠 Концепция
DiceMaster Pro — это не просто генератор случайных чисел. Это полноценный симулятор игральных костей для настольных игр, RPG и статистических экспериментов:

✅ Поддержка различных костей — d4, d6, d8, d10, d12, d20, d100 и пользовательские.

✅ Бросок нескольких костей одновременно — например, 3d6, 5d20.

✅ Взвешенные кости — возможность задать смещение вероятности для каждой грани (нечестная игра).

✅ История бросков — сохранение всех результатов с временными метками.

✅ Статистика — среднее значение, медиана, мода, распределение.

✅ Анимация броска (в консоли) — имитация "встряхивания" и выпадения результата.

✅ Экспорт истории в CSV/JSON.

✅ Интерактивный режим с командами или простой GUI.

🚀 Как запустить
Все версии работают в консоли и не требуют внешних библиотек (кроме стандартных). Инструкции:

bash
# Python
python dice_python.py

# C++
g++ -std=c++17 -O2 dice_cpp.cpp -o dice && ./dice

# Java
javac dice_java.java && java dice_java

# C#
dotnet run   # или csc dice_cs.cs && dice_cs.exe

# Go
go run dice_go.go

# Rust
cargo build --release && ./target/release/dice_rs

# JavaScript (Node.js)
node dice_js.js
